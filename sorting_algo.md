## Bubble Sort Implementation
Idea: Repeatedly swap adjacent elements if they are in the wrong order.

```python

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr
```
## 2. Selection Sort Implementation
Idea: Repeatedly find the minimum element from the unsorted part and move it to the sorted part.

```python

def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i + 1, n):
            if arr[j] < arr[min_index]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
```
## 3. Insertion Sort Implementation
Idea: Build the final sorted array one element at a time by comparing and inserting.

```python

def insertion_sort(arr):
    n = len(arr)
    for i in range(1, n):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr
```
